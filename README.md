# Exploratory Data Analysis of Superstore Sales Dataset using Python

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Superstore Sales dataset using Python. The objective of this analysis is to identify business insights, sales trends, profitability patterns, and the impact of discounts on overall business performance.

The project uses Python libraries such as Pandas, Matplotlib, and Seaborn for data cleaning, analysis, and visualization.

---

## 🎯 Objectives
- Analyze sales and profit trends
- Identify high-performing and loss-making products
- Understand the impact of discounts on profitability
- Perform regional and sub-category analysis
- Generate business insights using data visualization

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

---

## 📂 Dataset
The dataset used in this project is the Superstore Sales dataset containing:
- Orders
- Sales
- Profit
- Discounts
- Regions
- Product Categories
- Customer Information

---

## 📊 Exploratory Data Analysis Performed

### ✔ Data Understanding
- Dataset structure analysis
- Data types inspection
- Statistical summary

### ✔ Data Cleaning
- Missing value detection
- Duplicate value checking

### ✔ Sales Analysis
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Sales by Sub-Category

### ✔ Profit Analysis
- Profit by Category
- Profit by Sub-Category

### ✔ Relationship Analysis
- Discount vs Profit

---

## 🔍 Key Insights
- Technology category generated the highest sales and profit.
- Phones and Copiers were among the most profitable sub-categories.
- Tables generated significant losses despite high sales.
- Higher discount levels negatively impacted profitability.
- Sales showed an overall increasing trend from 2014 to 2017.

---

## 📈 Business Recommendations
- Optimize discount strategies to improve profitability.
- Focus marketing efforts on high-performing products.
- Investigate loss-making sub-categories like Tables and Bookcases.
- Continue using data-driven decision-making for business growth.
